# Data sources

## Urban Institute Education Data Portal API (CCD directory)

- General docs: `https://educationdata.urban.org/documentation/`
- School Districts (CCD directory): `https://educationdata.urban.org/documentation/school-districts.html`
- Schools (CCD directory): `https://educationdata.urban.org/documentation/schools.html`
- API FAQs: `https://urbaninstitute.github.io/education-data-faqs/`

We use the v1 API with the documented URL format:

`https://educationdata.urban.org/api/v1/{topic}/{source}/{endpoint}/{year}/...`
